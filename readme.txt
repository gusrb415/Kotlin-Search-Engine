Project Setup

Install IntelliJ IDEA in https://www.jetbrains.com/idea/
Install Gradle Build Tool in https://gradle.org/
Select open project: comp4321-Search-Engine-project

Run Spider

Open and Run SpiderMain.kt in the projet
File RocksDB is created.
